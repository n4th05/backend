# ✅ Checklist - Preparação para Entrevista Júnior

Use este checklist para garantir que você domina os conceitos essenciais antes de considerar-se preparado para vagas júnior.

## 📚 Fundamentos de Java

### Conceitos Básicos
- [ ] Sei o que são classes e objetos
- [ ] Entendo herança, polimorfismo, encapsulamento
- [ ] Conheço diferença entre interface e classe abstrata
- [ ] Sei usar Collections (List, Set, Map)
- [ ] Entendo exceções (try-catch-finally)
- [ ] Sei a diferença entre checked e unchecked exceptions
- [ ] Conheço tipos primitivos vs wrapper classes
- [ ] Entendo diferença entre == e equals()

### Java Moderno
- [ ] Sei usar Lambdas
- [ ] Entendo Streams API básico (map, filter, reduce)
- [ ] Conheço Optional
- [ ] Sei trabalhar com LocalDate/LocalDateTime

## 🍃 Spring Boot

### Conceitos Fundamentais
- [ ] Sei o que é Spring Boot e por que usar
- [ ] Entendo Inversion of Control (IoC)
- [ ] Entendo Dependency Injection (DI)
- [ ] Sei a diferença entre @Component, @Service, @Repository, @Controller
- [ ] Conheço @Autowired e quando usar
- [ ] Entendo Auto-Configuration
- [ ] Sei o que são Starters

### REST APIs
- [ ] Sei criar endpoints REST básicos (GET, POST, PUT, DELETE)
- [ ] Entendo diferença entre @RestController e @Controller
- [ ] Sei usar @PathVariable e @RequestParam
- [ ] Conheço códigos de status HTTP principais (200, 201, 400, 404, 500)
- [ ] Sei retornar ResponseEntity
- [ ] Entendo princípios REST

### Arquitetura
- [ ] Sei explicar arquitetura em camadas (Controller → Service → Repository)
- [ ] Entendo responsabilidade de cada camada
- [ ] Sei por que não retornar entidades JPA diretamente
- [ ] Conheço o conceito de DTOs

## 🗄️ Banco de Dados e JPA

### SQL Básico
- [ ] Sei fazer SELECT com WHERE, ORDER BY, LIMIT
- [ ] Entendo JOINs (INNER, LEFT, RIGHT)
- [ ] Sei usar GROUP BY e agregações (COUNT, SUM, AVG)
- [ ] Conheço constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE, NOT NULL)

### JPA/Hibernate
- [ ] Sei criar entidades JPA com @Entity
- [ ] Entendo @Id e @GeneratedValue
- [ ] Conheço validações Bean Validation (@NotNull, @NotBlank, @Size)
- [ ] Sei criar relacionamentos (@ManyToOne, @OneToMany)
- [ ] Entendo diferença entre EAGER e LAZY loading
- [ ] Sei o que é problema N+1 e como evitar
- [ ] Conheço JpaRepository e seus métodos prontos
- [ ] Sei criar query methods customizados

## ✅ Validação e Tratamento de Erros

### Validação
- [ ] Sei usar Bean Validation annotations
- [ ] Conheço @Valid e quando usar
- [ ] Sei validar dados no service layer
- [ ] Entendo diferença entre validação técnica e de negócio

### Exceções
- [ ] Sei criar exceções customizadas
- [ ] Conheço @ControllerAdvice e @ExceptionHandler
- [ ] Sei retornar mensagens de erro padronizadas
- [ ] Entendo como tratar erros globalmente

## 🧪 Testes

### Conceitos
- [ ] Entendo diferença entre testes unitários e de integração
- [ ] Sei escrever testes com JUnit 5
- [ ] Conheço Mockito para mockar dependências
- [ ] Sei usar @SpringBootTest
- [ ] Entendo conceito de cobertura de testes
- [ ] Sei testar controllers com MockMvc

### Prática
- [ ] Sei nomear testes descritivamente (shouldDoSomethingWhenCondition)
- [ ] Entendo padrão AAA (Arrange, Act, Assert)
- [ ] Sei testar casos de sucesso E falha
- [ ] Conheço assertions básicas

## 🔐 Segurança

- [ ] Sei proteger endpoints específicos
